# 📊 Netflix Movie Data Analysis

## 🚀 Project Overview
This project performs Exploratory Data Analysis (EDA) on a Netflix movie dataset to uncover insights about genres, popularity, and ratings.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Dataset Features
- Release Year
- Title
- Popularity
- Vote Count
- Vote Average
- Genre

## 📊 Project Workflow
1. Data Cleaning
2. Handling Missing Values
3. Genre Splitting & Exploding
4. Data Visualization
5. Insights & Conclusion

## 📈 Key Insights
- Identified most frequent movie genres
- Found highest rated movies based on vote average
- Analyzed popularity trends over time
